# 3060-Assignment-1

The project was to create a simple but effective website for data sources, info on collecting data and analysis methods, and showing publication venues. The website was aimed for researchers and PHD students, who may not have a ton of time to learn a complicated system.

The project is to have 4 sections:

# 1. DATA SOURCES Section 

Where we get data from. Each source should show: 

• Source name

 • What kind of data it provides
 
 • Brief description 

 # 2. COLLECTION METHODS Section
 
 How we collect the data.

 Each method should show:
 
 • Method name 
 
• What it's used for 

• Pros and cons

#  3. ANALYSIS METHODS Section 

How we analyze the data. Each method should show:

 • Method name 
 
 • Type of analysis 
 
 • Common use cases 

# 4. PUBLICATION VENUES Section 

Where to publish research. Each venue should show: 

• Venue name and abbreviation

 • Type (conference or journal) 
 
• Focus area 

• How relevant to MSR (high, medium) 


I took the liberty of adding one extra section, a home/landing page for newcomers to the website
